ActivityHTML

ActivityPub in HTML

by Charles Iliya Krempeaux

ActivityHTML uses HTML (rather than JSON-LD) as a wire-format for ActivityPub.

ActivityHTML puts ActivityPub information in HTML using RDFa (rather than JSON-LD or JSON).

RDFa lets you markup HTML with Linked Data name-spaces with something that feels a bit like Microformats.
